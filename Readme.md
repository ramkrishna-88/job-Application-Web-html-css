<h2><a class="anchor" id="Project Overview"></a>Project Overview</h2>

This project is a web-based job application form that collects user details through a structured interface.

- Steps performed by the application:
- Display a user-friendly job application form.
- Collect personal details like first name, last name, and email.
- Allow users to select a job role from a dropdown menu.
- Capture address information using a textarea field.
- Collect additional details such as city and pincode.
- Allow users to select a date.
- Enable users to upload their CV.
- Submit the form using the "Apply Now" button.
